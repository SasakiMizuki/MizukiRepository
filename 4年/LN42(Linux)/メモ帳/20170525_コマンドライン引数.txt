・コマンドライン引数
	…実行形式のファイル(プログラムやスクリプト)に対して渡すパラメータ
	
	…シェルスクリプトにコマンドライン引数を渡すには、スクリプト指定時に引数として指定する。
	
	ex)hoge.hoge@hoge>./hoge.sh 100 abc		// パスを指定。引数はspace値で渡す。
	
	…渡されたコマンドライン引数は、スクリプト内で以下の変数で処理する。
		・スクリプトファイル名 … $0
		・コマンドライン引数の数 … $#
		・コマンドライン引数一覧 … $*
		・n番目の引数の値 … $n

・シェルスクリプトでの演算処理
	…exprコマンドを使う全体を(`)で囲むので注意。
	【書式】
	`expr 値1 値2 値3`
	
	ex)num3 = `expr $num1 + $num2`


○if文

【書式】
if[ 条件式Ⅰ ]
then
	処理1
elif[ 条件式2 ]
then
	処理2
else
	処理3
fi

・[ 条件式 ]
   ↑     ↑ ここにスペースを忘れない。

・if、elifの下にはthenを記述。elseには書かない。

・elifはelse if。fiはendif


以下全部同じ結果
if test $n -eq 0

if [ $n = 0 ]

if [`expr $n = 0` = 1 ]

	…readコマンドを使うと入力待ちを実装できる。

【書式】
read 格納先変数
